DPM SafeSteps


DPM SafeSteps is a native iOS application that automatically detects walking, collects realtime gyroscope and location data, asks users to report fall risk hazards, and visualizes the data on a map.

This application is a part of the fall risk assessment and monitoring research project by Postdoctoral Fellow Dr. Hoonyong Lee and PhD Candidate John Sohn, sponsored by Liberty Mutual Insurance.


App Features:

  • Automatic walking detection using iPhone's motion data
  • Collects gyroscope and location data from wrist IMU sensor and iPhone's GPS sensor on Firebase for research
  • Allows users to report a fall hazard: type, intensity, location, photo, comments
  • Visualizes hazards and walking path on a map
