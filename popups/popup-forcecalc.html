Force Calculator


Force Calculator is an application that can calculate for any force-related formula from high school physics. For each formula, users can choose any variable to get a result in. The application also features a user-friendly UI design.

Some formulas include: Centripetal Force, Newton's Second Law, and Hooke's Law. There are 11 force-related formulas in total.


⚠️  Java is required to run the application. Download it here if not already installed.
